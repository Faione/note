# 四、事件流

## (1) 说明

|**事件**|**说明**|
|:-:|:-:|
|对象创建|创建对象接入到系统中|
|对象删除|删除对象并回收资源|
|对象更新|更新对象状态与资源分配|
|对象描述|获得对象的状态、资源描述|
|对象监控|上报对象的监控数据|

## (1) Agent启动流程

```plantuml
@startuml
skinparam roundCorner 25
skinparam responseMessageBelowArrow true

participant User as f1
participant Server as f2
participant Factory as f3
participant AgentCore as f4
participant RpcClient as f5
participant driver as f6
participant Collector as f7
participant CommonSDK as f8


f1    ->     f2  :收到请求
f2    ->     f3  :写入"对象创建"事件
f2    -->    f1  :返回"事件发布成功"
f3    ->     f4  :创建接入对象
f4    ->     f5  :请求GUID
f5   -->     f4  :返回GUID
f4    ->     f4  :创建对象上下文
f4   -->     f6  :调用dirver创建
f6    ->     f8  :调用底层SDK进行创建
f6   -->     f4  :返回结果
f4   -->     f4  :更新对象上下文
f4    ->     f5  :注册对象方法
f3    ->     f7  :创建收集协程
f7    ->     f8  :收集数据
f8    -->    f7  :返回数据
@enduml
```
