## 一、基础

**声明**

```
# 行公式排版
$ c = \sqrt{a^{2}+b_{xy}^{2}+e^{x}} $

# 块公式排版
$$ c = \sqrt{a^{2}+b_{xy}^{2} +e^{x}} $$
```

**字符标注**

```
\hat{x} : 加^号  

\overline{x} : 加横线

\widehat{x} : 加宽^

\widetilde{x} : 加波浪线 

\dot{x} : 加一个点

\ddot{x} : 加两个点 
```
