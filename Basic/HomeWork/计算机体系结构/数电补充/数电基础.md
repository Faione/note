# 数电基础

## 一、逻辑代数基础

### 基本逻辑门电路

**与**

$$Y = A\ AND\ B = A·B = AB$$

**或**

$$Y = A\ OR\ B = A+B$$

**非**

$$Y = NOT\ A = A^{'}$$

**与非**

$$Y = A\ NAND\ B = (AB)^{'}$$

**或非**

$$Y = A\ NOR\ B = (A+B)^{'}$$

**异或**

$$Y = A\ XOR\ B = A^{'}B + AB^{'} = A\oplus B$$
>X:Exclusive(排他地)

**同或**

$$Y = A\ XNOR\ B = A^{'}B^{'} + AB = A\odot B$$

### 常用逻辑公式

**摩根定理**

$$ (AB)^{'} = A^{'} + B^{'}\\ (A+B)^{'} = A^{'}B^{'}$$
